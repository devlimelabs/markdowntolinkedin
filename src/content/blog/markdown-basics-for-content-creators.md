## Quick Answer

Markdown is a simple way to format text using symbols. **Bold** uses `**text**`, *italic* uses `*text*`, and links use `[text](url)`. It works everywhere from GitHub to Notion to our LinkedIn formatter.

---

## What is Markdown?

Markdown is a lightweight markup language created by John Gruber in 2004. It lets you format text using simple symbols instead of clicking buttons or memorizing keyboard shortcuts.

**Why use Markdown?**
- Works in plain text files
- Portable across platforms
- Faster than rich text editors
- Easy to learn (5 minutes!)

## Essential Syntax

### Text Formatting

```markdown
**bold text**
*italic text*
~~strikethrough~~
`inline code`
```

Results in:
- **bold text**
- *italic text*
- ~~strikethrough~~
- `inline code`

### Headers

```markdown
# Heading 1
## Heading 2
### Heading 3
```

Use headers to structure your content. On LinkedIn, our tool converts these to bold text since LinkedIn doesn't support true headers.

### Lists

**Unordered lists:**
```markdown
- Item one
- Item two
- Item three
```

**Ordered lists:**
```markdown
1. First step
2. Second step
3. Third step
```

### Links

```markdown
[Click here](https://example.com)
```

Creates a clickable link: [Click here](https://example.com)

### Blockquotes

```markdown
> This is a quote
```

> This is a quote

### Images

```markdown
![Alt text](image-url.jpg)
```
